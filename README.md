# Doctor Appointment Management System

A web-based appointment management system built using HTML, CSS, JavaScript, PHP, and MySQL.

## 📌 Project Overview
This system allows patients to book doctor appointments online and enables admins to manage and track all appointments efficiently.

## ✨ Features
- Patient registration and login
- Book, view, and cancel appointments
- Admin dashboard to manage appointments
- Database-driven reporting and insights
- Responsive UI for all devices

## 🛠️ Technologies Used
| Technology | Purpose |
|---|---|
| HTML5 & CSS3 | Frontend UI |
| JavaScript | Client-side logic |
| PHP | Backend server logic |
| MySQL | Database management |

## 📂 Project Highlights
- Designed relational MySQL database for data storage and retrieval
- Built admin panel to track and manage appointment records
- Created SOPs and system documentation
- Debugged and resolved issues using systematic approach
